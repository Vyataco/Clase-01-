# Laboratorio 01: Landing Page

## Estructura Semántica
Para este proyecto he utilizado etiquetas de HTML5 semántico como `<header>`, `<main>`, `<section>` y `<footer>`. Esto permite que la página sea más fácil de leer para los navegadores y mejora la accesibilidad.

## Accesibilidad
He aplicado principios de accesibilidad (A11y) incluyendo:
- Atributos `alt` en todas las imágenes.
- Uso de `aria-label` en los enlaces de redes sociales para que los lectores de pantalla puedan identificarlos.

## Uso de IA (Prompting)
Utilicé a Gemini como asistente de IA para:
- **Estructura:** Pedí sugerencias para organizar las secciones de FAQ (Preguntas Frecuentes).
- **Depuración:** Me ayudó a corregir la ruta de las imágenes dentro de la carpeta `/img` y a configurar correctamente el Favicon.

## Logros Adicionales
- **Modo Oscuro:** Implementé un botón con JavaScript para cambiar los colores de la página.
- **Favicon:** Agregué un icono personalizado a la pestaña del navegador.
